title: nest help
sidebar_label: help

Show the global help or the help of a sub-command.

```shell script
nest help [...command]
```

## Arguments

 | argument   | description                |
 | ---------- | -------------------------- |
 | ...command | A command or a sub-command |

:::info
Show the global help if no command is provided
:::

## Examples

```shell script
nest help
nest help login
nest help publish 1.2.5 --pre
```
