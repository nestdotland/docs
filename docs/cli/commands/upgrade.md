Upgrade nest cli to the given version.

```shell script
nest upgrade [version]
```

## Arguments

 | argument | default | description            |
 | -------- | ------- | ---------------------- |
 | version  | latest  | A given semver version |

## Examples

```shell script
nest upgrade
nest upgrade 1.0.0
```
